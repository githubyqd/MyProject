# 模拟选手比赛
## 数据处理
	选手信息包含名字、分数
	用map容器存储参赛选手和选手的编号
	用vector容器存储相应比赛轮数的选手编号
	multimap容器存储小组分数比赛信息
	deque容器存储评分结果
## 比赛过程
	产生选手
	选手抽签
	选手比赛
	查看比赛结果
